# Sea Battle Bot

Telegram-бот для игры в Морской бой с соперником.

## Как играть

1. **Создать игру**: `/newgame` — бот выдаст код игры (например, `ABCDEF`)
2. **Пригласить соперника**: отправьте ему код
3. **Присоединиться**: `/join ABCDEF`
4. **Расставить корабли**: корабли расставляются автоматически. Нажмите « Переставить», если хотите другой вариант, а когда расстановка устраивает — «Подтвердить».
5. **Стрелять**: когда оба расставили корабли, ходите по очереди, нажимая на клетки доски соперника.

## Game modes

Поддерживаются два режима: соло (против бота) и мультиплеер (два игрока). В **обоих** режимах перед началом показывается экран броска кубика, определяющий, кто делает первый ход. При ничьей предлагается переброс. В соло вы бросаете свой кубик, а кубик бота подбрасывается автоматически — экран результата броска теперь появляется одинаково во всех играх, включая одиночные.

## Недавние изменения интерфейса

### Верхняя панель: Статистика и Настройки
В шапке приложения рядом с переключателем языка появились две кнопки: **Stats** (статистика) и **Settings** (настройки).

- ** Stats** — экран статистики: процент побед, победы/поражения/ничьи и последние игры с пометкой режима и соперника.
- ** Settings** — панель, где можно выбрать тему оформления ( Ocean / Forest), а также включить или выключить **Звук** и **Вибрацию**.

Эти настройки раньше были внутри каждой игры (отдельные панели темы/звука на поле); теперь они вынесены в общую панель Настроек и доступны из шапки в любой момент.

### Главное меню
Четыре игры — Морской бой, Покер в кости, Шашки, Нарды — выводятся сеткой 2×2. Кнопка **Присоединиться** (ввод кода игры) занимает всю ширину под ними. Статистика перенесена в верхнюю панель (кнопка ).

### Скрытый режим «На раздевание»
Режим «На раздевание» (Морской бой) больше не показывается в подменю Морского боя по умолчанию. Чтобы открыть его:

- в подменю Морского боя нажмите 10 раз подряд на пустую область под кнопкой **Выйти** (обычным темпом; длинная пауза между нажатиями сбрасывает счётчик);
- после 10-го нажатия кнопка режима появится.

Кнопка снова скрывается, как только вы выйдете из подменю, закончите игру или покинете приложение.

### Список активных игр
В списке активных игр у каждой партии теперь свой опознавательный эмодзи: морской бой, покер в кости, шашки, нарды.

### Доска нард
- Внешняя рамка доски стала компактнее.
- Подсказка «поверните телефон» убрана.
- Лоток вывода (bear-off / «дом»), куда снимаются выигранные фишки, теперь — компактная полоска справа от доски (оставлена, чтобы партию по-прежнему можно было выиграть снятием фишек).

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/newgame` | Создать новую игру |
| `/join XXXX` | Присоединиться по коду |
| `/leave` | Выйти из игры |
| `/cancel` | Отменить выбор при расстановке |

## Установка и запуск

### Локально

```bash
# 1. Клонировать репозиторий
cd sea_battle_bot

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Получить токен бота у @BotFather в Telegram

# 4. Вписать токен в config.py
# BOT_TOKEN = "ваш_токен"

# 5. Запустить
python main.py
```

### На сервере (Railway / Render / PythonAnywhere)

#### Railway (рекомендуется)

1. Залейте проект в GitHub
2. Создайте новый проект на [Railway](https://railway.app)
3. Подключите репозиторий
4. В настройках добавьте переменную окружения `BOT_TOKEN`
5. В `config.py` замените на:

```python
import os
BOT_TOKEN = os.getenv("BOT_TOKEN", "YOUR_BOT_TOKEN_HERE")
```

6. Railway сам установит зависимости и запустит `python main.py`

#### Render

1. Залейте проект на GitHub
2. Создайте новый Web Service на [Render](https://render.com)
3. Настройки:
 - Runtime: Python 3
 - Build Command: `pip install -r requirements.txt`
 - Start Command: `python main.py`
4. Добавьте переменную окружения `BOT_TOKEN`
5. Обновите `config.py` как указано выше для Railway

#### PythonAnywhere

1. Загрузите файлы через вкладку Files
2. Откройте Bash консоль
3. Установите зависимости: `pip install -r requirements.txt`
4. Настройте токен в `config.py`
5. Запустите: `python main.py` (будет работать пока открыта консоль)
6. Для фоновой работы используйте Always-on task (только на платных аккаунтах)
